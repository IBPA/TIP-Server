Instructions

Step 1: Download example.xlsx and open it. The first sheet, "Guide", includes an example and also some specification for each field.

Step 2: Open the second sheet, "Template", and fill compound and assay columns. If a compound has multiple assays, only fill assay columns for the following rows. See the example in "Guide" for details.

Step 3: Save only "Template" sheet as a .csv file. Note that a CSV file can only have one sheet.

Step 4: Upload your newly saved CSV file. If there are errors, you will be notified. Fix the errors and upload.
